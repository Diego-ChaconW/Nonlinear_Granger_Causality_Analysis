# Results Directory

This directory contains the outputs from each step of the pipeline.

## Subdirectories

### `hyperparameters/`
`.pkl` files from **Step 1** (hyperparameter search). Each file contains
the grid search results and the best hyperparameter configuration found.

Naming convention: `hyperparameters_{map}_{direction}.pkl`

### `experiments/`
`.pkl` files from **Step 2** (experiment execution). Each file contains
metrics from N random initializations of the causality test.

Naming convention: `results_{map}_{architecture}_{direction}_final.pkl`

### `figures/`
`.png` images from **Step 3** (results analysis). Includes histograms
comparing NAR vs NARX distributions and statistical plots.

## Note

These directories are initially empty. Results are generated by running
the pipeline (see `README.md` in the project root for instructions).
The `.gitignore` excludes `.pkl` and `.png` files to keep the repository
lightweight.
